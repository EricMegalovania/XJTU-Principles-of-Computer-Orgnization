# 单周期 CPU 设计

单周期 CPU 指的是一条指令的执行在一个时钟周期内完成，然后开始下一条指令的执行，即一条指令用一个时钟周期完成。

**地址总线**和**数据总线**均为 32 位

## 指令选取

从 MIPS-C 指令集中进行选择，因此每条指令为 32 位

### R-R 运算指令 10 条

```assembly
add    # 加
sub    # 减
and    # 与
or     # 或
xor    # 异或
sll    # 逻辑左移
srl    # 逻辑右移
sra    # 算术右移
slt    # 带符号小于置 1
sltu   # 无符号小于置 1
```

### R-I 运算指令 2 条

```assembly
addi   # 加立即数
ori    # 或立即数
```

### 分支指令 2 条

```assembly
beq    # 等于转移
j      # 无条件跳转
```

### 加载指令 1 条

```assembly
lw    # 加载字
```

### 存储指令 1 条

```assembly
sw    # 存储字
```
